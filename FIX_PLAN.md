# OneClickVirt 修复方案文档

## 问题清单

### 问题1：兑换码管理添加和批量生成失败
**问题描述**：管理后台菜单"兑换码管理"的添加和批量生成功能都提示失败。

**问题分析**：
1. 前端在提交时已经将元转换为分：`amount = form.value.amount * 100`
2. 后端在接收时可能存在数据类型或验证问题
3. 需要检查前后端数据结构是否一致

**修复方案**：
1. 修复后端兑换码创建接口，添加类型转换和异常处理
2. 批量生成接口增加金额单位自动检测和转换
3. 添加详细的日志记录，便于调试

---

### 问题2：订单金额显示不一致
**问题描述**：充值25元，订单中金额只显示0.25元。

**问题分析**：
- 后端存储金额单位为"分"，前端显示需要转换
- 可能存在重复转换或单位处理错误
- 数据库中存储的金额可能是错误的

**修复方案**：
1. 统一金额处理逻辑：后端统一用"分"存储，前端统一转换显示
2. 增加订单金额验证，防止错误金额写入
3. 添加未支付订单清理功能（手动删除+定时任务自动清理）

---

### 问题3：用户管理缺少代理商功能
**问题描述**：
1. 用户管理中缺少代理商级别类型
2. 代理商无法登录到自己权限对应的菜单
3. 缺少手动设置用户邮箱激活状态的功能

**修复方案**：
1. 在用户模型中添加代理商标记字段
2. 更新用户权限系统，支持代理商级权限
3. 在用户管理界面添加代理商标记和邮箱激活状态设置
4. 修复代理商登录逻辑，确保能访问正确的菜单

---

### 问题4：邮件发送功能无法使用
**问题描述**：测试及邮箱发送功能无法使用，即使配置了SMTP也发不了邮件。

**问题分析**：
1. SMTP配置可能不完整或格式错误
2. 邮件发送服务可能没有正确初始化
3. 可能缺少邮件发送相关的代码或依赖

**修复方案**：
1. 检查SMTP配置完整性（host、port、username、password）
2. 添加邮件发送测试接口
3. 修复邮件发送服务，增加错误处理和日志
4. 添加邮件发送状态监控

---

### 问题5：实名认证配置无法保存且缺少配置项
**问题描述**：
1. 系统配置中实名认证配置无法保存
2. 缺少实名认证的完整配置项
3. 实名管理菜单看不到用户提交的信息
4. 无法手动确认身份证验证信息

**修复方案**：
1. 完善实名认证配置数据结构和保存逻辑
2. 修复实名管理查询接口，确保能看到用户提交信息
3. 添加手动实名认证确认功能
4. 增加实名认证状态管理

---

### 问题6：实例申请失败（500错误）
**问题描述**：用户在申请领取实例时，配置好后点提交申请，提示"Request failed with status code 500"。

**问题分析**：
- 可能是后端处理逻辑出错
- 可能是数据库操作失败
- 可能是资源不足或权限问题

**修复方案**：
1. 检查实例申请相关后端代码
2. 增加详细的错误日志
3. 修复可能的数据库操作问题
4. 添加参数验证
