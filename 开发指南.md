# OneClickVirt 本地开发环境设置指南

- **项目名称**: OneClickVirt
- **项目路径**: C:\Users\admin\yuanqiai\oneclickvirt

## 项目结构
```
oneclickvirt/
├── server/          # Go后端服务
│   ├── main.go     # 后端入口文件
│   ├── config.yaml # 配置文件
│   └── ...
├── web/             # Vue3前端
│   ├── src/        # 源代码
│   └── package.json
├── start.bat        # 启动脚本
└── stop.bat         # 停止脚本
```

## 快速启动

### 方法1: 使用启动脚本（推荐）
双击运行 `start.bat` 文件，将自动启动前后端服务

### 方法2: 手动命令启动

**启动后端服务:**
```bash
cd C:\Users\admin\yuanqiai\oneclickvirt\server
go run main.go
```

**启动前端服务:**
```bash
cd C:\Users\admin\yuanqiai\oneclickvirt\web
npm run dev
```

## 开发说明

### 后端开发
- 语言: Go 1.26.1
- 框架: Gin
- 配置文件: server/config.yaml
- 端口: 8890

### 前端开发
- 框架: Vue 3
- 构建工具: Vite
- UI组件: Element Plus
- 端口: 5173

### 修改配置
主要配置文件位于: `server/config.yaml`
包含数据库、认证、支付等配置

## 常见问题

1. **端口被占用**
   - 修改 server/config.yaml 中的端口号
   - 或修改 web/vite.config.js 中的端口号

2. **依赖问题**
   - 后端: cd server && go mod download
   - 前端: cd web && npm install

3. **数据库连接失败**
   - 检查 config.yaml 中的 MySQL 配置
   - 确保MySQL服务已启动

## 二次开发建议

1. **修改前端界面**
   - 编辑 web/src 目录下的Vue组件

2. **添加后端API**
   - 在 server/api 目录添加新的API处理函数
   - 在 server/router 目录注册路由

3. **修改业务逻辑**
   - 在 server/service 目录修改业务逻辑
   - 在 server/model 目录修改数据模型
